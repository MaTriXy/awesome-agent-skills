Awesome Agent Skills

A curated collection of skills, tools, and capabilities for AI coding agents.

What Are Agent Skills?

Think of Agent Skills as instructions for AI assistants. Instead of the AI needing to know everything upfront, skills let it learn new abilities on the fly, like giving an assistant a new recipe card instead of a whole cookbook.

Skills are simple text files called SKILL.md that teach an AI how to perform specific tasks. When you ask the AI to do something, it finds the right skill, reads the instructions, and gets to work.

How It Works

  • Browse: The AI reviews the available skills.
  • Load: When needed, it reads the detailed instructions.
  • Use: The AI follows the steps and executes the task flawlessly.

How to Use Skills

Claude.ai

Click the skill icon in the chat interface. Add skills from the marketplace or upload custom skills. Claude will activate them automatically when relevant.

Google Antigravity

Antigravity supports Workspace Skills (located in /.agent/skills/) and Global Skills (located in ~/.gemini/antigravity/skills).

VS Code / GitHub Copilot

Store your skills in directories containing a SKILL.md file under .github/skills/.

OpenAI Codex

Use the built-in $skill-creator or install new skills from directories using the $skill-installer.

How to Create a Skill

Skills are straightforward markdown files. Below is the basic structure you need to get started.

---
name: my-skill-name
description: A clear description of what this skill does.
---

# My Skill Name

Detailed description of the skill's purpose.

## When to Use This Skill
- Use case 1
- Use case 2

## Instructions
[Detailed instructions for the agent on how to execute this skill]

Create a directory with the name of your skill, and place this SKILL.md file inside it. You can optionally include helper scripts or document templates.
